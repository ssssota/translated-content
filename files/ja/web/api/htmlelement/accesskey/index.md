**`HTMLElement.accessKey`** プロパティは、ユーザーが押すとこの要素へジャンプすることができるキーストロークを設定します。

> **Note:** `HTMLElement.accessKey` プロパティは、ブラウザーにすでに存在するキーバインドとよく衝突するため、めったに使われることはありません。これを回避するために、ブラウザーは accesskey の動作を他の「修飾」キー（<kbd>Alt</kbd> + accesskey など）と一緒に押された場合に動作するよう実装しています。
