{{APIRef("HTML DOM")}}

**`HTMLTableElement.caption`** プロパティは表のキャプションを表します。この表に関連付けられた caption 要素がない場合は、このプロパティは `null` になります。

## 構文

```js
var string = tableElement.caption;
```

## 例

```js
if (table.caption) {
  // キャプションに対して何かを行う
}
```
