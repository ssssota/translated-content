{{HTMLRef}}

HTML <legend> 元素用于表示其父元素 {{HTMLElement("fieldset")}} 的内容标题。

{{EmbedInteractiveExample("pages/tabbed/legend.html", "tabbed-standard")}}

内容分类 无。
允许的内容 语句内容（Phrasing content）。
标签忽略 {{no_tag_omission}}
允许的父元素 {{HTMLElement("fieldset")}} ，并且<legend>作为第一个子元素
Implicit ARIA role No corresponding role
Permitted ARIA roles No role permitted
DOM 接口 {{domxref("HTMLLegendElement")}}

属性


这个元素只有全局属性。
