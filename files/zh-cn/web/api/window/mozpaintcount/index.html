{{APIRef}}

返回当前文档渲染到屏幕上所需的时间。

语法

var paintCount = window.mozPaintCount;
  • paintCount 保存 window.mozPaintCount 属性值。
  • window.mozPaintCount 的值是长整数, 文档是新建的时候从0开始, 每渲染一次增加1。

说明

不是任何W3C技术规范或建议的一部分。
