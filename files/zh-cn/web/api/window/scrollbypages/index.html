概述

在当前文档页面按照指定的页数翻页.

语法

window.scrollByPages(pages)

参数

  • pages 翻页数,正数向下翻,负数向上翻.

例子

// 当前文档向下翻一页
window.scrollByPages(1);

// 当前文档向上翻一页
window.scrollByPages(-1);

备注

查看 window.scrollBy, window.scrollByLines, window.scroll, window.scrollTo.

规范

DOM Level 0 不属于任何标准
