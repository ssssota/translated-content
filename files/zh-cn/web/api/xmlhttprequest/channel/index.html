创建请求的时候， XMLHttpRequest.channel  是一个被对象使用的  nsIChannel 。如果管道（channel） 还没被创建的话，它的值是 null 。在一个 multi-part  请求案例中，它是初始化的管道，不是 multi-part 请求中的不同部分。

需要权限提升。
