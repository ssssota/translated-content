HTML <input type="datetime"> 是一个控件，用于输入日期和时间（小时、分钟、秒和秒的小数）以及时区。这个特性已经 从 WHATWG HTML 中移除，并且不再受浏览器支持。

反之，浏览器实现了（并且提倡开发者使用）{{HTMLElement("input/datetime-local", "datetime-local")}} 输入类型。
